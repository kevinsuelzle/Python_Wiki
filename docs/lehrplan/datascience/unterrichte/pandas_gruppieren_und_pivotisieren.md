## Gruppieren in Pandas

Das Gruppieren von Daten ist eine der wichtigsten Aufgaben in der Datenanalyse. Es erm√∂glicht die Organisation von Daten in kategorische Segmente, wodurch die Analyse komplexer Daten vereinfacht wird. Die Gruppierung in Pandas wird mit der groupby-Methode erreicht. Diese Methode teilt die Daten in Gruppen auf, die einen gemeinsamen Wert in einer bestimmten Spalte haben.

Zum Beispiel, wenn man eine Datenmenge mit Verkaufszahlen hat, k√∂nnte man diese Daten nach dem Verkaufsregion gruppieren. ### Beispiel 2: Gruppieren und Anwenden mehrerer Aggregatfunktionen


```python
import pandas as pd
```

### Beispiel 3: Pivot-Tabelle erstellen


```python
import pandas as pd

# Erstellen eines Beispiel-Datenrahmens
df = pd.DataFrame({
    'Datum': ['2023-01-01', '2023-01-01', '2023-01-02', '2023-01-02'],
    'Produkt': ['Apfel', 'Banane', 'Apfel', 'Banane'],
    'Verkaufsmenge': [10, 15, 20, 25]
})
print(df, "\n")

# Erstellen einer Pivot-Tabelle
pivot = df.pivot_table(values='Verkaufsmenge',
                       index='Datum',
                       columns='Produkt',
                       aggfunc='sum')
print(pivot)
```

In diesem Beispiel erstellen wir eine Pivot-Tabelle aus dem DataFrame `df`. Wir setzen 'Datum' als Index, 'Produkt' als Spalten und summieren die 'Verkaufsmenge'. Das Ergebnis zeigt die Verkaufsmengen f√ºr √Ñpfel und Bananen getrennt nach Datum.

### Beispiel 5: Pivotisieren mit Multi-Index
